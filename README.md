# SmolWooby

**Requirements**
1. [Node.js](https://nodejs.org/en/)
2. [Angular CLI](https://cli.angular.io/)
3. [ghpages](https://github.com/angular-schule/angular-cli-ghpages)

**Build**
1. `npm install`
2. `ng serve --open`

**Deploy**
1. `ng build --prod --base-href https:// username .github.io/ reponame /`
2. `ngh`
